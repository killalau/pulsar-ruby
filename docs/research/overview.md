# Research Overview

This folder contains research notes for the Ruby Apache Pulsar client project.

## Contents

- [Official Apache Pulsar Links](official-links.md): Curated official references
  for client libraries, protocol work, APIs, and feature comparison.
- [Architecture Patterns](architecture-patterns.md): Shared client architecture
  patterns and early Ruby architecture implications.
- [Feature Matrix](feature-matrix.md): Cross-client feature support matrix,
  starting with the Java baseline.
- [MVP Scope](mvp-scope.md): Proposed first implementation scope for the Ruby
  client.
- [Official Client Research Plan](official-client-research-plan.md): Repeatable
  plan for analyzing each official client and producing a final comparison.
- [Official Client Comparison](official-client-comparison.md): Per-client
  analysis notes and final cross-client comparison.
- [Local Client Repositories](local-client-repos.md): Local workspace convention
  and repository list for cloned official client source references.
- [Protocol Notes](protocol-notes.md): Binary protocol findings for lookup,
  producer, consumer, ack, and flow-control behavior.
- [Ruby Design Considerations](ruby-design-considerations.md): Ruby-specific API,
  concurrency, dependency, packaging, and implementation decisions.
- [Ruby Client Landscape](ruby-client-landscape.md): Initial assessment of
  existing Ruby Pulsar clients and their maintenance status.
